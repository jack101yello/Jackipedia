# Open Set

In [[Topology]], an open subset on a topological space $(X, \mathcal{T})$ is an element of the topology $\mathcal{T}$ on that space. Its complement is called a [[Closed Set]].

## Category Theory

In [[Category Theory]], one can construct a [[Functor]] from the category of topological spaces to [[The Category of Sets]] such that every topological space is mapped to the set of its open subsets.
