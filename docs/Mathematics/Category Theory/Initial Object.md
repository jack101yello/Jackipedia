# Initial Object

In [[Category Theory]], an object $i$ in a [[Category]] $C$ is said to be initial if for every $c \in C$, there exists a unique morphism $i \to c$. The converse notion is a [[Terminal Object]].

## Examples of Initial Objects

1. In [[The Category of Sets]], the empty set is initial.
2. Likewise in $\text{Top}$.
3. In a preorder, an initial object is a global minimum.
