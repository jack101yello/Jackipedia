# Pre-Sheaf

A pre-sheaf is a [[Contravariant Functor]] which is a [[Set-Valued Functor]]. That is to say, a pre-sheaf $F$ on a [[Category]] $C$ is a [[Functor]] $F: C^{\text{op}} \to \text{Set}$.

## Sheaves

Let $X$ be a set and $F: \mathcal{O}(X)^{\text{op}} \to \text{Set}$ be a pre-sheaf on the [[Poset Category]] of $X$, $\mathcal{O}(X)$. $F$ is a sheaf if it preserves all [[Covering]] [[Colimit]]s.
