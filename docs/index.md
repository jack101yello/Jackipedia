
# Jackipedia

Inspired by the nLab, I wanted a place to consolidate my notes on topics throughout math and physics, which would allow me to quickly reference ideas I've already come across when I need them in the future. To this end, I created Jackipedia.

While this is really intended for personal use, I hope anyone who stumbles upon this wiki finds it helpful. It's in a very rudimentary stage at the moment, but I hope to add to it as time goes on.
